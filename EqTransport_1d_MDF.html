quation de Transport 1D:

L'quation de transport avec vitesse $(\alpha)$ est donne par :

$\frac{\partial u}{\partial t} + \alpha \frac{\partial u}{\partial x} = 0 $

Paramtres de l'exercice

On pose $\alpha = 1$.

  1. Pour la condition initiale : $u_0(x) = \sin(x)$, calculer la solution analytique du problme.
  2. Implmenter la mthode des diffrences finies sur ce problme avec les schmas :
    • dcentr gauche,
    • dcentr droite,
    • et celui de Lax-Wendroff.
  3. Reprsenter sur un mme graphique les solutions discrtes et exactes.

Solution analytique

La solution analytique du problme avec la condition initiale $u_0(x) = \sin(x)$ est : $ u(x, t) = \sin(x - \alpha t)$ o $\alpha = 1$. Donc $u(x, t) = sin(x - t)$

Implmentation des schmas de diffrences finies

Schma dcentr gauche

Le schma dcentr gauche utilise une approximation de la drive spatiale en prenant le point courant et le point prcdent : $\frac{\partial u}{\partial x} \approx \frac{u_i^n - u_{i-1}^n}{\Delta x}$

En substituant cette approximation dans l'quation de transport, nous obtenons : $\frac{u_i^{n+1} - u_i^n}{\Delta t} + \alpha \frac{u_i^n - u_{i-1}^n}{\Delta x} = 0$

En rarrangeant pour trouver $u_i^{n+1}$ : $u_i^{n+1} = u_i^n - \frac{\alpha \Delta t}{\Delta x} (u_i^n - u_{i-1}^n)$

Schma dcentr droite

Le schma dcentr droite utilise une approximation de la drive spatiale en prenant le point courant et le point suivant : $\frac{\partial u}{\partial x} \approx \frac{u_{i+1}^n - u_i^n}{\Delta x}$

En substituant cette approximation dans l'quation de transport, nous obtenons : $\frac{u_i^{n+1} - u_i^n}{\Delta t} + \alpha \frac{u_{i+1}^n - u_i^n}{\Delta x} = 0$

En rarrangeant pour trouver $u_i^{n+1}$ : $ u_i^{n+1} = u_i^n - \frac{\alpha \Delta t}{\Delta x} (u_{i+1}^n - u_i^n)$

Schma de Lax-Wendroff

Le schma de Lax-Wendroff utilise une combinaison de termes de diffrence centre et de termes de diffrence temporelle pour amliorer la prcision.

  1. Approximation de la drive temporelle :

$\frac{\partial u}{\partial t} \approx \frac{u_i^{n+1} - u_i^n}{\Delta t}$

  1. Approximation de la drive spatiale centre : $\frac{\partial u}{\partial x} \approx \frac{u_{i+1}^n - u_{i-1}^n}{2 \Delta x}$

  2. Approximation du terme de diffusion : $\frac{\partial^2 u}{\partial x^2} \approx \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{\Delta x^2}$

En substituant ces approximations dans l'quation de transport, nous obtenons : $\frac{u_i^{n+1} - u_i^n}{\Delta t} + \alpha \frac{u_{i+1}^n - u_{i-1}^n}{2 \Delta x} = \frac{\alpha^2 \Delta t}{2 \Delta x^2} (u_{i+1}^n - 2u_i^n + u_{i-1}^n)$

En rarrangeant pour trouver $u_i^{n+1}$ : $ u_i^{n+1} = u_i^n - \frac{\alpha \Delta t}{2 \Delta x} (u_{i+1}^n - u_{i-1}^n) + \frac{\alpha^2 \Delta t^2}{2 \Delta x^2} (u_{i+1}^n - 2u_i^n + u_{i-1}^n)$

Explications

  1. Solution Analytique : La solution exacte est calcule avec $u(x,t) = \sin(x - \alpha t)$.
  2. Schma Dcentr Gauche : Utilise les valeurs prcdentes pour mettre jour les valeurs futures.
  3. Schma Dcentr Droite : Utilise les valeurs futures pour mettre jour les valeurs prsentes.
  4. Conditions Priodiques : Les valeurs aux extrmits du domaine sont mises jour pour simuler un domaine priodique.

// Paramtres
L = 2 * %pi;          // Longueur du domaine
T = 2 * %pi;          // Temps total de simulation
nx = 100;             // Nombre de points spatiaux
nt = 100;             // Nombre de points temporels
dx = L / nx;          // Pas spatial
dt = T / nt;          // Pas temporel
alpha = 1;            // Vitesse de propagation

// Discrtisation spatiale et temporelle
x = linspace(0, L, nx);
t = linspace(0, T, nt);

// Condition initiale
u0 = sin(x);

// Schma dcentr  gauche
u_left = u0;
for n = 1:nt
    u_left(2:nx) = u_left(2:nx) - alpha * dt / dx * (u_left(2:nx) - u_left(1:nx-1));
    u_left(1) = u_left($); // Condition priodique
end

// Schma dcentr  droite
u_right = u0;
for n = 1:nt
    u_right(1:nx-1) = u_right(1:nx-1) - alpha * dt / dx * (u_right(2:nx) - u_right(1:nx-1));
    u_right($) = u_right(1); // Condition priodique
end

// Solution analytique
u_exact = sin(x - alpha * T);

// Tracer les solutions
clf(); // Effacer la figure prcdente
plot(x, u_exact, 'k-', 'LineWidth', 3);  // Solution analytique
plot(x, u_left, 'r--', 'LineWidth', 2);  // Schma dcentr  gauche
plot(x, u_right, 'b-.', 'LineWidth', 2); // Schma dcentr  droite

// Lgendes et titres
legend("Solution exacte", "Dcentr  gauche", "Dcentr  droite"); // Ajouter des lgendes aux courbes
xlabel('x'); // tiqueter l'axe des x
ylabel('u'); // tiqueter l'axe des y
title("Solutions de l''quation de transport"); // Titre du graphique

// Paramtres
alpha = 1; // Coefficient de transport
L = 2 * %pi; // Longueur du domaine
Nx = 100; // Nombre de points spatiaux
x = linspace(0, L, Nx); // Grille spatiale
Delta_x = x(2) - x(1); // Pas spatial
Delta_t = 0.01; // Pas temporel
T = 2; // Temps total de simulation
Nt = round(T / Delta_t); // Nombre de pas temporels

// Condition initiale
u_exact = sin(x); // Solution exacte  t=0
u_left = u_exact; // Solution pour le schma dcentr  gauche
u_right = u_exact; // Solution pour le schma dcentr  droite
u_lax_wendroff = u_exact; // Solution pour le schma de Lax-Wendroff

// Boucle temporelle
for n = 1:Nt
    // Schma dcentr  gauche
    for i = 2:Nx
        u_left(i) = u_left(i) - (alpha * Delta_t / Delta_x) * (u_left(i) - u_left(i-1));
    end
    
    // Schma dcentr  droite
    for i = 1:Nx-1
        u_right(i) = u_right(i) - (alpha * Delta_t / Delta_x) * (u_right(i+1) - u_right(i));
    end
    
    // Schma de Lax-Wendroff
    for i = 2:Nx-1
        u_lax_wendroff(i) = u_lax_wendroff(i) - (alpha * Delta_t / (2 * Delta_x)) * (u_lax_wendroff(i+1) - u_lax_wendroff(i-1)) + ...
                             (alpha^2 * Delta_t^2 / (2 * Delta_x^2)) * (u_lax_wendroff(i+1) - 2 * u_lax_wendroff(i) + u_lax_wendroff(i-1));
    end
    
    // Conditions aux limites (priodiques)
    u_left(1) = u_left(Nx);
    u_right(Nx) = u_right(1);
    u_lax_wendroff(1) = u_lax_wendroff(Nx);
    u_lax_wendroff(Nx) = u_lax_wendroff(2); // Pour maintenir la condition priodique
end

// Solution exacte  t=T
u_exact_final = sin(x - alpha * T);

clf; // Effacer la figure prcdente

plot(x, u_left, 'b--', 'LineWidth', 1.5); // Schma dcentr  gauche
plot(x, u_right, 'g--', 'LineWidth', 1.5); // Schma dcentr  droite
plot(x, u_lax_wendroff, 'm--', 'LineWidth', 1.5); // Schma de Lax-Wendroff
